# Algorithmic Complexity  
We analyze the complexity using:

$$
T(n) = O(n \log n)
$$

More examples:

- Searching: \(O(\log n)\)  
- Sorting: \(O(n \log n)\)  
- Brute force: \(O(2^n)\)

# Code Example

```python
def compute_complexity(n):
    # Example O(n log n) function
    total = 0
    for i in range(n):
        j = 1
        while j < n:
            total += 1
            j *= 2
    return total
```
